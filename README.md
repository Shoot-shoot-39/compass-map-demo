\# 地理院地図 + コンパス + 偏角補正



このページはスマホのコンパスと位置情報を使って、

あなたが向いている方向を地理院地図上に表示します。



\## 特徴

\- コンパス方向をリアルタイムに取得

\- 日本の地域ごとにおおまかな偏角を自動計算（±1°以内）

\- GitHub Pagesで動作（HTTPS対応）



\## 使い方

1\. このリポジトリをGitHubにアップロード  

2\. 「Settings → Pages → Branch: main → /(root)」を選択  

3\. 公開URLにアクセス  

4\. 「方位センサー許可」を押して許可  

5\. 地図上にあなたの向きの線が表示されます



💡 iPhoneのSafariでは  

設定 → Safari → 「モーションと画面の向きへのアクセスを許可」をオンにしてください。
